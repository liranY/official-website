Flash QT Desktop Wallet


Install Core Wallet
  1. Download the latest version of the Flash Coin Core Wallet (https://www.flashcoin.io/#qt-wallet). Select the correct wallet for your OS (Operating System): Windows, Linux or OSX.
  2. Run the Installer.
  3. You can use the default settings or change them to your liking.
  4. Click Install and then wait for the Core Wallet to install.
  5. Run the Flash Coin Core.
  6. The wallet will take a while to sync and download the full blockchain. However, you may speed up the process by manually downloading the Bootstrap.dat file as explained in the steps below.

Syncing the Wallet Quickly
  1. Stop the Wallet (most likely it is not running, so ignore “error: couldn’t connect to server”).
  2. Open your Search bar on your desktop and type %appdata% then go into the Flashcoin folder. (C:\Users\User_name\AppData\Roaming\Flashcoin)
    • Windows: ~/AppData/Flashcoin
    • OSX: ~/Library/Application Support/ Flashcoin
    • Linux: ~/.Flashcoin
  3. Delete the Blocks folder, Chainstate folder and peer.dat file.

  4. Download the Flash Blockchain snapshot from the following link: (https://flashcoin.io/download/flashcoin_blockchain_snapshot_23jan2018.tar.gz)

    Download WinRAR and install from the following link: https://winrar.en.softonic.com/

  5. Right click the Flashcoin Blockchain file you just downloaded (flashcoin_blockchain_snapshot_23jan2018.rar) and click extract. Extract the files to a known folder. In the extracted folder you will see Blocks, Chainstate, Database folders and Peers.dat in the new folder.
  6. Go to the %appdata% flashcoin folder you previously deleted the old files and copy and paste the 3 folders and peers.dat file to the flashcoin folder.
  7. You’re all done! The wallet should be accessible within 5-20 minutes, depending on how old the bootstrap is.
